This project is an AI-powered advertisement optimization system that helps users generate the best advertising strategy based on their input.

The user provides:

Target audience 👥
Product name 🛍️

Using Gemini AI, the system analyzes the input and suggests:

📅 Best time to run the advertisement
📝 Ad description/content
🎯 Targeting strategy
💡 Features
AI-generated advertisement content
Smart audience targeting suggestions
Best timing recommendations for ads
Easy-to-use interface
Real-time response using Gemini API
🛠️ Tech Stack
Frontend: (React / HTML / CSS / JS )
Backend: (Python)
AI Model: Gemini API
Database: (MongoDB)
⚙️ How It Works
User enters product name and audience type
Request is sent to backend
Gemini AI processes the input
AI returns:
Best ad timing
Ad description
